# CSB-project1

## Requirements:

To run: Django, Python

You may need werkzeug and pwnedpasswords to test the fixes.

## Installation:

1. Clone the repo.

2. Via the command line, go inside the forum directory.

3. Run `python3 manage.py runserver`
